CSS3 Mastery

From basic styling to advanced layout and animations

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of HTML documents. CSS3, the latest version, introduces powerful features like Grid, Flexbox, animations, and custom properties that enable modern web design.

Key Concept: CSS separates content from presentation, allowing you to style multiple HTML pages with a single stylesheet while maintaining consistency and efficiency.

The CSS Box Model

Understanding the box model is fundamental to CSS layout. Every element is a rectangular box with content, padding, border, and margin.

Content

Yellow: Content | Green: Padding | Blue: Border | Transparent: Margin

.box-model-demo {
    width: 200px;           /* Content width */
    height: 100px;          /* Content height */
    padding: 20px;          /* Space inside border */
    border: 10px solid #007bff; /* Border thickness and style */
    margin: 20px;           /* Space outside border */
    background: #ffc107;    /* Background color */

    /* Box-sizing controls how dimensions are calculated */
    box-sizing: border-box; /* Include padding and border in dimensions */
}

Box-Sizing Property:

content-box (default)

Width and height apply only to content. Padding and border are added to the total size.

border-box

Width and height include content, padding, and border. More predictable for layout.

CSS Grid Layout

CSS Grid is a powerful 2-dimensional layout system that allows precise control over both rows and columns.

Header
Sidebar
Main
Footer
.grid-demo {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
    grid-template-areas: 
        "header header"
        "sidebar main"
        "footer footer";
    gap: 1rem;
    padding: 1rem;
}

.grid-item {
    background: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%);
    color: white;
    padding: 2rem;
    text-align: center;
    border-radius: 10px;
    font-weight: 600;
}

/* Advanced Grid Example */
.advanced-grid {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;
    grid-template-rows: auto 1fr auto;
    grid-gap: 20px;
    min-height: 100vh;
}

Grid Properties Explained:

  • display: grid: Creates a grid container
  • grid-template-columns: Defines column sizes and number
  • grid-template-rows: Defines row sizes and number
  • gap: Sets spacing between grid items
  • grid-template-areas: Names grid areas for semantic layout

Flexbox Layout

Flexbox is perfect for 1-dimensional layouts, providing flexible alignment and distribution of space.

Item 1
Item 2
Item 3
.flex-demo {
    display: flex;
    justify-content: space-between; /* Horizontal alignment */
    align-items: center;            /* Vertical alignment */
    gap: 1rem;
    padding: 1rem;
    flex-wrap: wrap;               /* Allow wrapping */
}

.flex-item {
    background: linear-gradient(135deg, #f7b733 0%, #fc4a1a 100%);
    color: white;
    padding: 1rem 2rem;
    border-radius: 25px;
    font-weight: 600;
    flex: 1;                       /* Grow to fill space */
    text-align: center;
    min-width: 120px;
}

/* Flexbox Properties */
.flex-container {
    display: flex;
    flex-direction: row;           /* row | column */
    justify-content: flex-start;   /* main axis alignment */
    align-items: stretch;          /* cross axis alignment */
    flex-wrap: nowrap;            /* wrap | nowrap */
}

Flexbox vs Grid:

Use Flexbox for:
  • Navigation bars
  • Centering content
  • Distributing space evenly
  • One-dimensional layouts
Use Grid for:
  • Complex page layouts
  • Two-dimensional control
  • Overlapping elements
  • Magazine-style layouts

CSS Custom Properties (Variables)

CSS variables enable dynamic theming and reduce code duplication by storing reusable values.

Dynamic Theme Example

This card uses CSS variables for theming. Change the color above to see it update!

/* Define CSS Variables in :root */
:root {
    --primary-color: #007bff;
    --secondary-color: #6c757d;
    --success-color: #28a745;
    --border-radius: 15px;
    --box-shadow: 0 10px 30px rgba(0, 0, 0, 0.1);
    --transition: all 0.3s ease;
}

/* Use Variables Throughout CSS */
.btn-primary {
    background-color: var(--primary-color);
    border-color: var(--primary-color);
    border-radius: var(--border-radius);
    transition: var(--transition);
}

.card {
    border-radius: var(--border-radius);
    box-shadow: var(--box-shadow);
}

/* JavaScript can update CSS variables */
document.documentElement.style.setProperty('--primary-color', '#dc3545');

Benefits of CSS Variables:

  • Dynamic theming without recompiling
  • Reduced code duplication
  • JavaScript can modify values in real-time
  • Scoped to specific elements or global
  • Better maintainability

Responsive Design

CSS media queries and flexible layouts ensure websites work across all devices and screen sizes.

/* Mobile First Approach */
.responsive-grid {
    display: grid;
    grid-template-columns: 1fr;
    gap: 1rem;
}

/* Tablet */
@media (min-width: 768px) {
    .responsive-grid {
        grid-template-columns: 1fr 1fr;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .responsive-grid {
        grid-template-columns: repeat(3, 1fr);
    }
}

/* Container Queries (Modern) */
@container (min-width: 400px) {
    .card {
        display: flex;
        flex-direction: row;
    }
}

/* Responsive Typography */
.responsive-text {
    font-size: clamp(1rem, 2.5vw, 2rem);
}

/* Print Styles */
@media print {
    .no-print { display: none; }
    body { font-size: 12pt; }
}

CSS Architecture and Best Practices

BEM Methodology
/* Block */
.card { }

/* Element */
.card__header { }
.card__body { }
.card__footer { }

/* Modifier */
.card--featured { }
.card--large { }
.card__header--dark { }
CSS Organization
/* 1. Variables */
:root { --primary: #007bff; }

/* 2. Base styles */
* { box-sizing: border-box; }

/* 3. Layout */
.container { max-width: 1200px; }

/* 4. Components */
.btn { padding: 0.5rem 1rem; }

/* 5. Utilities */
.text-center { text-align: center; }

Performance Best Practices:

  • Minimize CSS file size
  • Use efficient selectors
  • Avoid deep nesting
  • Leverage CSS caching
  • Critical CSS for above-fold content
  • Use CSS containment
  • Optimize for paint and layout
  • Consider CSS-in-JS for components

CSS3 Summary

CSS3 transforms static HTML into beautiful, interactive web experiences. From layout systems like Grid and Flexbox to animations and custom properties, CSS3 provides the tools for modern web design.

Key takeaways:

  • Box Model: Understanding spacing and sizing fundamentals
  • Layout Systems: Grid for 2D layouts, Flexbox for 1D layouts
  • Animations: Bringing interfaces to life with CSS animations
  • Variables: Dynamic theming with CSS custom properties
  • Responsive Design: Creating layouts that work on all devices
  • Performance: Writing efficient, maintainable CSS
